# Webpack3.x+React16.x+Cli
这是一个基于 Webpack3.x + React16.x + React-router + Redux 的脚手架

下载并运行

// 克隆代码
git clone https://github.com/colien/Webpack3.x-React16.x-Cli.git

// 进入项目目录
cd Webpack3.x-React16.x-Cli

// 安装依赖
npm install

// 生成 dll 文件
npm run dll

// 本地启动
npm run local

// 测试环境打包
npm run dev

// 生成环境打包
npm run build
